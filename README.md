# XCLIP-AREA

## Dependencies.

- Tesseract OCR.
- xclip, xterm, xdotool.
- ImageMagick (import tool).

## Help

xclip-area

    Usage: xclip-area
    
    Select an area on the screen, process the image with an OCR engine
    (tesseract) and copy the text.

xclip-edit

    Usage: xclip-edit
    
    Edit selected text with a text editor.

xclip-type

    Usage: xclip-type
    
    Type the text on the clipboard.
